# Claude Recepten Generator

Deze folder bevat een recepten generator skill. Met deze skill kun je snel en eenvoudig recepten creëren op basis van verschillende ingrediënten en voorkeuren.

## Hoe te gebruiken

1. Clone deze repository naar je lokale machine.
2. Installeer en configureer Claude op je lokale machine.
3. Open Claude in deze folder.
4. Start de recepten generator skill door `/recipe-generator` in te voeren en volg de instructies om je gewenste recept te genereren.
